### KyOSC

Based on Ahmet Kizilay's [AndrOSC](https://github.com/ahmetkizilay/AndrOSC)

An app for Android to use OSC from your device 
with the ability to design graphical templates 
within the app. 

#### Available Controllers
* OSC_Button 
* OSC_Toggle
* OSC_Pad
* OSC_VSlider
* OSC_HSlider
